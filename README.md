# webpack-express-boilerplate
For startup projects.

### Prerequisites
- node express
- webpack
- eslint
- babel
- react

Clone project
``` sh
$ git clone https://github.com/Rebecca-chu/webpack-express-boilerplate.git
```

Install packages
``` sh
$ yarn install
```

### Development stage
``` sh
$ yarn dev
```

### Production
``` sh
$ yarn build
$ yarn start
```
