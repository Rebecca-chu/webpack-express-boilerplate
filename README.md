# webpack-express-boilerplate

## Prerequisites
- node express
- webpack
- eslint
- babel
- react

Clone project
``` sh
$ git clone https://github.com/Rebecca-chu/webpack-express-boilerplate.git
```

Install packages
``` sh
$ yarn add
```

### Development stage
``` sh
$ yarn dev
```

### Production
``` sh
$ yarn build
$ yarn start
```
